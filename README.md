# 我的首个Web3 DApp

## 功能特性
- 🔗 MetaMask 钱包连接/断开
- 🌐 多RPC节点容错配置
- 💰 实时查询 ETH 余额
- 🪙 实时查询 ERC20 代币余额（USDC）
- 🎨 响应式UI设计，支持移动端

## 技术栈
- **前端框架**: Next.js 14 + TypeScript
- **样式方案**: Tailwind CSS
- **Web3开发**: Wagmi + Viem
- **钱包连接**: MetaMask, OKX Wallet
- **开发网络**: Sepolia测试网


# Web3 DApp 钱包应用
